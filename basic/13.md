## while 文

while 文はインデントされているところの処理を、条件式が True の間処理を実行する構文です。

<script type="py-editor">
# 無限に ぴえん と唱え続ける
while True:
    print('ぴえん')
</script>

<script type="py-editor">
# 一度も実行しない
spam = False
while spam:
    print('spam')
</script>


<script type="py-editor">
# 100 以上を数えるまで実行する
count = 0
while count < 100:
    count = count + 1
    print(count)
</script>


## for 文

for 文を用いると、リスト等から 1 つずつデータを取り出して、インデントされているところの処理を実行することができます。

<script type="py-editor">
spam = [1,2,3,4,5]

for ham in spam:
    print(ham)
</script>
